# CSCI 335 Machine Learning Project - Group 2

## Abstract
This repository contains the machine learning project for CSCI 335. The project demonstrates the application of machine learning techniques to solve real-world problems, showcasing implementation of various ML algorithms, data processing pipelines, and model evaluation methodologies.

## How to Run the Project

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bcc2761/csci-335-ml.git
   cd csci-335-ml
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

Instructions for running the project will be added as implementations are completed.

- `code/` - All scripts, notebooks, and implementations
- `resources/` - Research papers, reference links, notes, and presentation slides
